
Реализация экранной клавиатуры на WinForms с использованием GDI+.

В проекте показан подход к отрисовке UI без стандартных контролов, обработка нажатий и передача ввода через SendKeys.

Текущая версия основана на более раннем материале (2017) и существенно переработана с учётом практического опыта.
